# Screenshot Capture Guide

Screenshots needed for the README. Save each as a PNG in `docs/images/`.

## How to capture

- Use `Cmd+Shift+4` on macOS to select a region
- Aim for ~800px wide, crop tightly to the relevant UI element
- Use a dark theme for consistency (the extension is designed for dark themes)
