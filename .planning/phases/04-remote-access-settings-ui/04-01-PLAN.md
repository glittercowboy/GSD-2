<objective>
Create the two core API endpoints that the Remote Access settings panel will consume: a password change endpoint at `/api/settings/password` (NOT under `/api/auth/*` to avoid the auth exemption) and a Tailscale status endpoint.

Purpose: The RemoteAccessPanel (Plan 03) needs backend routes to change passwords and check Tailscale status. These must exist before the UI can be wired up.

- Phase 1 creates `src/web/web-auth-storage.ts` with `setPassword(password, dir?)` that hashes via scrypt and stores in `~/.gsd/web-auth.json`, then calls `rotateSessionSecret()` to invalidate all sessions.
- Phase 1 creates `src/web/web-session-auth.ts` with `createSessionToken(secret)` and `getOrCreateSessionSecret(dir?)`.
- Phase 1 creates `GET /api/auth/status` returning `{ configured: boolean, authenticated: boolean }`.
- Phase 1 creates `/api/auth/*` routes exempt from credential checks (but with Origin validation). This is WHY we place password change under `/api/settings/` instead — it MUST require authentication.
- Phase 2 creates `src/web/tailscale.ts` with `getTailscaleStatus()`, `isTailscaleInstalled()`.
- Phase 1 extends `authFetch` in `web/lib/auth.ts` to work with both bearer tokens (localhost) and session cookies (Tailscale HTTPS).

    `export async function POST(request: Request): Promise<Response>`:

    1. Parse request body: `const { newPassword } = await request.json() as { newPassword: string }`
    2. Validate: if `!newPassword || typeof newPassword !== 'string' || newPassword.length < 4` return `Response.json({ error: "Password must be at least 4 characters" }, { status: 400 })`
    3. Store password and rotate secret:
       ```typescript
       await setPassword(newPassword)
       // setPassword() internally:
       // - hashes with scrypt
       // - stores in ~/.gsd/web-auth.json
       // - calls rotateSessionSecret() which updates ~/.gsd/web-session-secret
       // - updates process.env.GSD_WEB_SESSION_SECRET with the new secret
       ```
    4. Re-issue session cookie for current browser (per D-07):
       ```typescript
       // Read the new secret (setPassword already updated process.env)
       const newSecret = await getOrCreateSessionSecret()
       const token = createSessionToken(newSecret)
       const cookieStore = await cookies()
       cookieStore.set("gsd-session", token, {
         httpOnly: true,
         secure: true,
         sameSite: "strict",
         maxAge: 30 * 24 * 60 * 60, // 30 days
         path: "/",
       })
       ```
    5. Return `Response.json({ ok: true })`

    Wrap the entire body in try/catch. On error: return `Response.json({ error: "Failed to update password" }, { status: 500 })`.

    Per D-06: No current password verification required. Single-user tool — if the user can call this endpoint, they are already authenticated by the proxy.

    Create `web/app/api/tailscale/status/route.ts`:

    ```typescript
    import { getTailscaleStatus, isTailscaleInstalled } from "../../../../src/web/tailscale"

    export const runtime = "nodejs"
    export const dynamic = "force-dynamic"

    export async function GET(): Promise<Response> {
      try {
        const installed = await isTailscaleInstalled()
        if (!installed) {
          return Response.json({
            installed: false,
            connected: false,
            hostname: "",
            tailnetUrl: "",
            dnsName: "",
          })
        }

        const info = await getTailscaleStatus()
        if (!info || !info.connected) {
          return Response.json({
            installed: true,
            connected: false,
            hostname: "",
            tailnetUrl: "",
            dnsName: "",
          })
        }

        return Response.json({
          installed: true,
          connected: info.connected,
          hostname: info.hostname,
          tailnetUrl: info.tailnetUrl,
          dnsName: info.dnsName,
        })
      } catch {
        return Response.json({
          installed: false,
          connected: false,
          hostname: "",
          tailnetUrl: "",
          dnsName: "",
        })
      }
    }
    ```

    interface TailscaleStatusResponse {

      dnsName: string       // e.g., "mybox.tailnet.ts.net" (trailing dot already stripped by Phase 2's tailscale.ts)
    }
