## What Was Built

**GsdLogo SVG swap:** Changed `src` from `/assets/gsd-2-mission-control-logo.png` to `/assets/gsd-logo-2000-transparent.svg`. The SVG file was already present in `public/assets/`. Single-line change; no other modifications needed.

**SingleColumnView project name header bar:** Added `projectName?: string` prop to `SingleColumnViewProps` interface. Added `FolderOpen` import from lucide-react. Added conditional header bar rendered above all views:
```tsx
{projectName && (
  <div className="flex items-center gap-2 px-4 py-1.5 border-b border-navy-600 bg-navy-900 shrink-0">
    <FolderOpen className="h-3 w-3 text-slate-500" />
    <span className="text-xs font-mono text-slate-400 truncate">{projectName}</span>
  </div>
)}
```
The `<main>` element was updated from `h-full overflow-hidden` to `flex flex-col` to support the header + view stack, and the inner div updated to `flex-1 min-h-0 overflow-hidden`.

**AppShell projectName derivation:** Extracted `const projectName` before the dashboard return block, deriving it as:
- `activeProjectPath`: extract basename, fall back to `state?.projectState?.milestone_name`
- No `activeProjectPath`: use `state?.projectState?.milestone_name`

Both `<Sidebar>` and `<SingleColumnView>` now receive `projectName={projectName}`, ensuring the Sidebar bordered section and the right panel header bar always show the same value.

**globals.css Firefox scrollbar support:** Added `scrollbar-width: thin` and `scrollbar-color: #334155 transparent` as base CSS properties on the `.scrollbar-thin` class, ahead of the existing webkit pseudo-element rules.

**Sidebar + SettingsView scrollbar-thin:** Added `scrollbar-thin` class and Firefox inline style to the flex-1 body scroll containers in both components.

**layout.test.tsx rewrite:** Removed the `TabLayout` import (deleted in a prior branch commit). Replaced TabLayout tests with SingleColumnView projectName source-text assertions and AppShell projectName derivation assertions.
